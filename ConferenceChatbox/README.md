# WorkshopConnect

## Overview

WorkshopConnect is a Java-based Workshop Management System developed to manage college workshops efficiently.

The system provides:

* Host Authentication
* Student Registration
* Attendance Tracking
* Workshop Chat System
* Anonymous Question Submission
* Polling System
* Activity Logging
* Workshop Statistics
* Automated Workshop Reports

---

## How To Run

1. Open terminal inside src folder
2. Compile:

javac MainServer.java

3. Run:

java MainServer
